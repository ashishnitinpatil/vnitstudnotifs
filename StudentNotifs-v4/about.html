VNIT Student Notifications Checker

This is a web app developed by Ashish Nitin Patil aka !mmorta!.
It periodically checks the VNIT Student Notifications website for any new notifications.
It tweets the new updates/notifications through it's twitter account VNITStudNotifs
It then posts the tweet on the FaceBook page - VNITStudNotifs through an IFTTT recipe.
The web app is powered by the Google App Engine platform.
It runs on Python & uses the Tweepy python library for tweeting.
